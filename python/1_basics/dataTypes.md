Python data types can be categorized based on **ordering** and **mutability**.

| Category  | Description                            | Data Type   |
|-----------|----------------------------------------|-------------|
| Ordered   | Elements maintain insertion order      | List, Tuple, String       |
| Unordered | Elements do not maintain order         | Set, Dictionary           |
| Immutable | Value cannot be changed after creation | int, float, string, tuple |
| Mutable   | Value can be modified after creation   | list, set, dictionary     |
