# Basic Arithmetic Operations

Python supports standard arithmetic operations.

| Operator | Operation | Example |
|--------|-----------|--------|
| `+` | Addition | `5 + 3 = 8` |
| `-` | Subtraction | `5 - 3 = 2` |
| `*` | Multiplication | `5 * 3 = 15` |
| `/` | Division | `5 / 3 = 1.67` |
| `%` | Modulus (Remainder) | `5 % 3 = 2` |
| `**` | Power (Exponent) | `5 ** 3 = 125` |

### Example

```python
a = 10
b = 2

print(a + b)   # 12  Addition
print(a - b)   # 8   Subtraction
print(a * b)   # 20  Multiplication
print(a / b)   # 5   Division
print(a % b)   # 0   Remainder
print(a ** b)  # 100 Power
````
