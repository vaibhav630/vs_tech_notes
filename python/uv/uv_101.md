# 🚀 UV (Python Package & Project Manager) – Complete Guide

## 📌 What is UV?

**UV** is a fast Python package and project manager (written in Rust) that replaces:

* `pip`, pip-tools, pipx, poetry, pyenv, twine
* virtual environments & more
* `requirements.txt`

⚡ It can be **up to 100x faster** than pip in some cases.

---

# 🛠️ Installation

## Install using pip

```bash
pip install uv
```

## Mac/Linux (alternative)

```bash
pip3 install uv
```

## Verify Installation

```bash
uv
```

### ✅ Output

```
Usage: uv [OPTIONS] COMMAND [ARGS]...
...
```

# 🐍 Python Version Management

## List available Python versions

```bash
uv python list
```

### ✅ Output

```
Installed:
  3.11.x
  3.10.x

Available:
  3.12.x
  3.9.x
  ...
```

---

## Install a Python version

```bash
uv python install 3.8
```

### ✅ Output

```
Downloading Python 3.8...
Installing...
Done.
```

---

## Find installed version

```bash
uv python find 3.8
```

### ✅ Output

```
/path/to/python3.8
```

---

## Uninstall Python version

```bash
uv python uninstall 3.8
```

# ▶️ Running Python Scripts

## Basic run

```bash
uv run main.py
```

### Example `main.py`

```python
import sys
print(sys.version)
```

### ✅ Output

```
3.11.x (default Python)
```

---

## Run with specific Python version

```bash
uv run --python 3.9.21 main.py
```

### ✅ Output

```
3.9.21
```

# 📦 Running with Dependencies (No Install Needed)

## Example (missing module error)

```bash
uv run --python 3.9 main.py
```

### ❌ Output

```
ModuleNotFoundError: No module named 'rich'
```

---

## Run with dependency

```bash
uv run --python 3.9 --with rich main.py
```

### ✅ Output

```
(Program runs successfully)
```

---

## Multiple dependencies

```bash
uv run --with rich --with requests main.py
```

# 🧠 Script Configuration (Auto Setup)

## Initialize script with config

```bash
uv init --script main.py --python 3.9.21
```

### Adds to top of script:

```python
# requires-python = "==3.9.21"
```

---

## Add dependencies to script

```bash
uv add --script main.py rich
uv add --script main.py requests
```

### ✅ Script gets updated:

```python
# dependencies = ["rich", "requests"]
```

---

## Run script (auto installs deps)

```bash
uv run main.py
```

# ⚠️ Handling Version Issues

## Example: TensorFlow error

```bash
uv add tensorflow
```

### ❌ Output

```
No compatible version for Python 3.13
```

---

## Fix by changing Python version

Edit `pyproject.toml`:

```toml
requires-python = ">=3.10"
```

Then:

```bash
uv add tensorflow
```
