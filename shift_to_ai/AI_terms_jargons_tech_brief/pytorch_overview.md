**PyTorch** is an open-source machine learning framework used to build and train AI models, especially in areas like deep learning, computer vision, and natural language processing.

It was developed by Meta AI 

---

## 🧠 Simple Explanation

Think of PyTorch as a **toolkit for teaching computers to learn from data**.

You use it to:

* Build neural networks (like brain-inspired models)
* Train models on data (images, text, numbers)
* Run predictions (AI inference)

---

## ⚙️ Key Features

### 1. Dynamic Computation Graph (Flexible)

* You can change the model structure on the fly
* Easier debugging compared to older frameworks

### 2. Python-Friendly

* Feels like normal Python code (very popular with developers)

### 3. GPU Acceleration

* Uses CUDA to run heavy computations fast on GPUs

### 4. Strong Ecosystem

Includes libraries like:

* **TorchVision** → image tasks
* **TorchText** → text/NLP
* **TorchAudio** → audio processing

---

## 🧩 Basic Example

```python
import torch

# Create a tensor
x = torch.tensor([1.0, 2.0, 3.0])

# Simple operation
y = x * 2

print(y)
```

👉 Output:

```
tensor([2., 4., 6.])
```

## ⚔️ PyTorch vs TensorFlow

| Feature     | PyTorch 🧠 | TensorFlow 🔧    |
| ----------- | ---------- | ---------------- |
| Ease of use | Very easy  | Slightly complex |
| Debugging   | Simple     | Harder           |
| Flexibility | High       | Medium           |
| Production  | Improving  | Strong           |
