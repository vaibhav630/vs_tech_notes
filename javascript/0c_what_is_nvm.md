**NVM** stands for **Node Version Manager**.

It’s a tool that lets you **install, manage, and switch between multiple versions of Node.js** on the same machine easily.

---

### 🔧 Why NVM is useful

When working with different projects, each might need a different Node.js version. Instead of uninstalling/reinstalling Node every time, NVM handles it cleanly.

---

### 🚀 What you can do with NVM

* Install multiple Node versions

  ```bash
  nvm install 18
  nvm install 20
  ```

* Switch between versions

  ```bash
  nvm use 18
  ```

* Set default version

  ```bash
  nvm alias default 20
  ```

* List installed versions

  ```bash
  nvm ls
  ```

---

### 🧠 Simple analogy

Think of NVM like a **remote control for Node.js versions** — instead of changing the TV, you just switch the channel.

---
