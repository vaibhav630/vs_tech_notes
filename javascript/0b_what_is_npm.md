**npm** stands for **Node Package Manager**.

It is a tool that comes bundled with Node.js and is used to **install, manage, and share JavaScript packages (libraries)**.

---

### 🔧 What npm does

* 📦 **Installs packages** (like React, Express, etc.)
* 🔄 **Manages dependencies** for your project
* 🚀 **Runs scripts** (like starting a server or building code)
* 🌐 Provides access to the huge **npm registry** (online collection of packages)

---

### 📌 Simple Example

If you want to use a library like Express.js:

```bash
npm install express
```

This downloads Express into your project.

---

### 📁 Key Concepts

* **package.json** → File that tracks your project dependencies
* **node_modules/** → Folder where packages are installed
* **npm scripts** → Custom commands (like `npm start`)

---

### ⚡ Common npm Commands

* `npm init` → Create a new project
* `npm install` → Install dependencies
* `npm install <package>` → Install a specific package
* `npm start` → Run your app (if defined)

---

### 🧠 In short:

**npm = App store + package manager for JavaScript developers**
